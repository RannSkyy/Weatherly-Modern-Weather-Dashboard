# 🌤 Weatherly – Modern Weather Dashboard

Weatherly is a responsive front-end web application that displays real-time weather data using the OpenWeather API. The project demonstrates DOM manipulation, asynchronous JavaScript (fetch API), and modern UI design using Flexbox.

## 🚀 Features

- Real-time weather data
- Search weather by city name
- Clean and modern user interface
- Responsive layout
- Error handling for invalid city or API issues
- Async/Await implementation

## 🛠 Tech Stack

- HTML5
- CSS3 (Flexbox)
- JavaScript (DOM + Fetch API)
- OpenWeather API
